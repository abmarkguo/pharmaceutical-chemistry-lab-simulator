| 吸入危害 | 类别1 | ⚠️ | 烃类/松节油 | 防止误吸 |

### 1.3 化学品存储原则

```
存储兼容性矩阵：

         | 酸 | 碱 | 氧化剂 | 还原剂 | 易燃 | 毒性
---------|----|----|--------|--------|------|-----
酸       | ✓  | ✗  | ✓     | ✗      | ✗    | ✓
碱       | ✗  | ✓  | ✓     | ✗      | ✗    | ✓
氧化剂   | ✓  | ✓  | ✓     | ✗      | ✗    | ✓
还原剂   | ✗  | ✗  | ✗     | ✓      | ✓    | ✓
易燃     | ✗  | ✗  | ✗     | ✓      | ✓    | ✓
毒性     | ✓  | ✓  | ✓     | ✓      | ✓    | ✓

✓ = 可共存  ✗ = 不可共存
```

**特殊存储要求**：
- 剧毒品：双锁双管/双人领取/专用柜/台账
- 易制毒/易制爆：公安备案/限量购买/使用登记
- 自燃物：惰性气氛/油封/专用存储
- 过氧化物形成物(THF/乙醚)：加BHT/检测过氧化物/避光

## 2. PPE个人防护装备

### 2.1 PPE选择矩阵
| 风险等级 | 眼部 | 手部 | 身体 | 呼吸 | 特别要求 |
|----------|------|------|------|------|----------|
| 1级-低 | 护目镜 | 丁腈手套 | 白大褂 | — | — |
| 2级-中 | 护目镜/面屏 | 丁腈/氯丁手套 | 白大褂+围裙 | 通风橱 | — |
| 3级-高 | 全面罩 | 双层手套(丁腈+氯丁) | 防化围裙+袖套 | 通风橱+防毒面具 | 紧急冲淋可达 |
| 4级-极高 | 正压面罩 | 防化手套+内层棉手套 | 全身防化服 | 正压呼吸器 | 隔离操作/应急待命 |

### 2.2 手套选择指南
| 手套材质 | 适用化学品 | 不适用 | 穿透时间 |
|----------|-----------|--------|----------|
| 丁腈 | 油类/弱酸碱/多数有机溶剂 | 强氧化剂/部分酮类 | 30-60min |
| 氯丁 | 广谱化学品 | 部分芳烃 | 60-120min |
| 丁基橡胶 | 酮类/强酸/气体 | 烃类 | 120-240min |
| PVA(聚乙烯醇) | 芳烃/氯代烃 | 水/醇类 | 120-240min |
| Viton(氟橡胶) | 芳烃/氯代烃/酸 | 酮类/酯类 | 120-240min |
| 乳胶 | 水溶液/弱酸碱 | 有机溶剂 | 10-30min |

**重要提醒**：
- 无任何手套能防护所有化学品
- 手套有穿透时间，长时间使用后需更换
- 手套破损立即更换
- 脱手套时避免接触污染面

## 3. 应急预案

### 3.1 化学品泄漏应急

```
泄漏响应流程：

小量泄漏（<100mL）
├─ 1. 评估危险性（查阅MSDS）
├─ 2. 佩戴适当PPE
├─ 3. 通风（开窗/开排风）
├─ 4. 吸附处理
│   ├─ 酸性→碳酸氢钠中和
│   ├─ 碱性→柠檬酸中和
│   ├─ 有机→专用吸附垫/蛭石
│   └─ 氧化性→还原性吸附剂
├─ 5. 收集至专用废物容器
└─ 6. 清理区域/记录报告

大量泄漏（≥100mL或有毒有害）
├─ 1. 立即疏散区域人员
├─ 2. 报告安全负责人
├─ 3. 关闭火源/电源
├─ 4. 专业人员处置
└─ 5. 事后调查报告
```

### 3.2 化学灼伤应急
| 部位 | 应急措施 | 注意事项 |
|------|----------|----------|
| 皮肤-酸 | 大量清水冲洗≥15min → 5%碳酸氢钠冲洗 | 禁止用强碱中和 |
| 皮肤-碱 | 大量清水冲洗≥15min → 2%硼酸冲洗 | 禁止用强酸中和 |
| 皮肤-HF | 大量清水冲洗 → 葡萄糖酸钙凝胶涂抹 | HF可穿透皮肤→骨坏死 |
| 眼睛 | 洗眼器冲洗≥15min → 就医 | 不要揉眼睛/保持眼睑张开 |
| 吸入 | 移至空气新鲜处 → 必要时吸氧 → 就医 | 保持呼吸道通畅 |
| 食入 | 不要催吐 → 饮水/牛奶稀释 → 就医 | 携带MSDS就医 |

### 3.3 火灾应急
| 火灾类型 | 灭火器选择 | 禁止使用 |
|----------|-----------|----------|
| A类-固体 | 水/泡沫/干粉 | — |
| B类-液体 | 泡沫/干粉/CO2 | 水（扩散火势） |
| C类-气体 | 干粉/CO2 | 水（除非冷却） |
| D类-金属 | 干沙/NaCl/Met-L-X | 水/CO2/泡沫 |
| 电气 | CO2/干粉 | 水/泡沫 |

**金属钠/钾火灾**：干沙或NaCl覆盖，绝对禁止用水和CO2

### 3.4 应急设备配置
| 设备 | 位置要求 | 维护要求 |
|------|----------|----------|
| 紧急冲淋装置 | 化学实验室出口15m内 | 每周测试/每年检定 |
| 洗眼器 | 每个实验台10m内 | 每周冲洗/每月检定 |
| 灭火器 | 每个房间至少1个 | 每月检查/每年充装 |
| 急救箱 | 实验室固定位置 | 定期检查补充 |
| 应急呼吸器 | 高危实验室 | 定期检查 |
| 泄漏应急包 | 化学品存储区 | 定期检查 |

## 4. 废物处理规范

### 4.1 废液分类与处理
| 类别 | 包含内容 | 收集容器 | 处理方式 | 法规依据 |
|------|----------|----------|----------|----------|
| 含卤有机废液 | 二氯甲烷/氯仿/四氯化碳 | 棕色玻璃瓶 | 委托焚烧 | HW06 |
| 非卤有机废液 | 甲醇/乙醇/丙酮/乙醚 | 棕色玻璃瓶 | 委托焚烧 | HW06 |
| 含酸废液 | 硫酸/盐酸/硝酸 | 耐酸塑料桶 | 中和后排放 | GB8978 |
| 含碱废液 | 氢氧化钠/氢氧化钾 | 塑料桶 | 中和后排放 | GB8978 |
| 含重金属废液 | 汞/铅/铬/镉 | 塑料桶 | 沉淀固化 | HW17/HW21 |
| 含氰废液 | 氰化物 | 单独收集 | 碱性氯化法 | HW33 |
| 含汞废液 | 汞盐/汞溶液 | 单独密封 | 硫化物沉淀 | HW29 |

### 4.2 废液处理操作规范
1. **分类收集**：不同类别废液严禁混装
2. **标签完整**：标明成分/浓度/日期/收集人
3. **兼容性检查**：混合前确认不会产生剧毒气体/爆炸
4. **暂存管理**：密封存放/通风良好/定期转移
5. **转移处置**：委托有资质单位处理/保留联单

### 4.3 固废处理
| 类型 | 处理方式 |
|------|----------|
| 空试剂瓶 | 清洗→玻璃回收/按废液类别处理残留 |
| 沾染化学品的纱布/滤纸 | 按化学品性质分类→危废 |
| 废催化剂 | 回收金属/委托处理 |
| 废色层柱 | 按硅胶/氧化铝→固废 |
| 动物尸体 | 冷冻→委托焚烧 |
| 生物废物 | 高压灭菌→黄色垃圾袋→焚烧 |

## 5. 动物实验伦理

### 5.1 3R原则详细实施

#### Replacement决策流程
```
实验需求
├─ 能否用计算机模拟？
│  ├─ ADME预测 → 用商业软件替代部分PK筛选
│  ├─ QSAR毒性预测 → 替代部分急性毒性筛选
│  └─ 分子对接 → 替代部分药效筛选
├─ 能否用体外方法？
│  ├─ 细胞实验 → 替代细胞毒性/肝毒性筛选
│  ├─ 组织培养 → 替代部分器官功能研究
│  ├─ 器官芯片 → 前沿替代（多器官交互）
│  └─ 体外代谢 → 肝微粒体替代部分PK
├─ 能否用低等生物？
│  ├─ 斑马鱼 → 替代部分发育毒性/心血管
│  ├─ 线虫 → 替代部分神经毒性/衰老研究
│  └─ 果蝇 → 替代部分遗传学实验
└─ 必须用哺乳动物
   → 进入Reduction和Refinement流程
```

#### Reduction实施
- 预实验确定效应大小→精确计算样本量
- 拉丁方/交叉设计减少个体差异
- 一只动物多终点（血液+组织+病理）
- 利用历史对照数据
- 共享对照组

#### Refinement实施
- 适当麻醉镇痛（异氟烷/戊巴比妥/氯胺酮-赛拉嗪）
- 术后镇痛（布比卡因/美洛昔康）
- 饲养丰容（垫料/筑巢材料/玩具/群居）
- 人道终点（不等到死亡才终止）
- 操作培训减少应激
- 安乐死方法（CO2渐近法/颈椎脱臼-小鼠/戊巴比妥过量-大鼠）

### 5.2 安乐死方法
| 方法 | 适用动物 | 优点 | 注意事项 |
|------|----------|------|----------|
| CO2渐近法 | 啮齿类 | 快速/无痛苦 | 充气速率20-30%V/min |
| 颈椎脱臼 | 小鼠(<200g) | 快速/无需设备 | 需培训认证 |
| 戊巴比妥过量 | 所有种属 | 可靠/平静 | 剂量≥100mg/kg iv/ip |
| 异氟烷过量 | 小型动物 | 快速/可逆前 | 需废气排除 |
| KCl iv（麻醉下） | 大型动物 | 心脏停搏 | 必须在深麻醉下 |

**禁止的安乐死方法**：
- 空气栓塞（除麻醉下大型动物）
- 氯仿/乙醚（毒性/安全隐患）
- 甲醛浸泡（非安乐死，活体禁用）
- 窒息/ drowning
- 低温（除胚胎/新生鼠）

## 6. 生物安全

### 6.1 生物安全级别
| 级别 | 病原体 | 实验室要求 | 操作要求 |
|------|--------|-----------|----------|
| BSL-1 | 无已知疾病风险 | 开放台面/洗手 | 基础PPE |
| BSL-2 | 中等风险（乙肝/沙门菌） | 生物安全柜/高压灭菌 | BSC II级+PPE |
| BSL-3 | 严重/致命（结核/炭疽） | 负压实验室/双门高压灭菌 | BSC+正压防护服 |
| BSL-4 | 致命无治疗（埃博拉） | 气密实验室/正压服 | 最高级别防护 |

### 6.2 生物安全柜使用
| 类型 | 保护对象 | 适用 |
|------|----------|------|
| BSC I级 | 操作者 | 低风险微生物 |
| BSC II级A2 | 操作者+产品+环境 | BSL-2常规使用 |
| BSC II级B2 | 操作者+产品+环境 | 挥发性化学品+生物 |
| BSC III级 | 最高级防护 | BSL-3/4高危病原 |

**BSC使用规范**：
1. 使用前5min开启/紫外消毒15min
2. 工作面分区：清洁区/操作区/污染区
3. 物品从清洁区→操作区→污染区单向移动
4. 手臂缓慢进出避免气流紊乱
5. 工作区后部格栅不可遮挡
6. 使用后紫外消毒15min/表面消毒
