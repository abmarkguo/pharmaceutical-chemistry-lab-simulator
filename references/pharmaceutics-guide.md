## 2. 片剂制备实验

### 2.1 工艺路线选择

```
API性质评估
├─ 流动性好 → 直接压片
├─ 流动性中等 → 干法制粒
└─ 流动性差/剂量小 → 湿法制粒

湿法制粒决策树：
├─ API热稳定？ → 否：考虑干法或直接压片
├─ API溶于水？ → 否：用水或水-醇制粒
├─ API与粘合剂相容？ → 否：更换粘合剂
└─ 含量均匀度风险？ → 是：增加混合时间/等量递加
```

### 2.2 处方设计

#### 常用辅料速查
| 功能 | 常用辅料 | 用量范围 | 注意事项 |
|------|----------|----------|----------|
| 填充剂 | 乳糖/淀粉/微晶纤维素/甘露醇 | 20-80% | 乳糖不适用于遗传性乳糖不耐受 |
| 粘合剂 | PVP/HPMC/淀粉浆/明胶溶液 | 2-10% | PVP K30常用，浓度5-10% |
| 崩解剂 | 交联CMC-Na/交联PVP/淀粉/L-HPC | 2-8% | 内加+外加（内外各50%） |
| 润滑剂 | 硬脂酸镁/微粉硅胶/滑石粉 | 0.5-2% | 硬脂酸镁过量影响崩解 |
| 助流剂 | 微粉硅胶/滑石粉 | 0.1-1% | 改善流动性 |
| 着色剂 | 氧化铁/色素 | 0.01-0.1% | 符合食品/药品标准 |

### 2.3 湿法制粒工艺模拟

```
阶段1: 处方计算
  → 决策点：每片重量的确定（API+辅料总量）
  → 辅料比例分配是否合理？

阶段2: 原料预处理
  → 粉碎/过筛（80-100目）
  → 决策点：粉碎度对溶出的影响

阶段3: 混合
  → 等量递加法混合（含量小API）
  → 决策点：混合时间是否充足？
  → V型混合器 vs 三维混合器

阶段4: 制粒
  → 粘合剂加入方式（喷雾/倾倒）
  → 制软材标准："手握成团，轻压即散"
  → 决策点：软材湿度是否合适？
  → ⚠️ 过湿→硬颗粒/干燥不透；过干→不成粒

阶段5: 干燥
  → 流化床/烘箱干燥
  → 干燥温度40-60°C（热敏性API降低）
  → 决策点：含水量是否达标（1-3%）？
  → ⚠️ 过度干燥→静电/颗粒碎裂

阶段6: 整粒
  → 过筛整粒（18-24目）
  → 决策点：颗粒粒度分布是否合理？

阶段7: 总混
  → 加入外加崩解剂+润滑剂
  → 混合时间5-15min
  → 决策点：润滑剂混合时间是否过长？
  → ⚠️ 硬脂酸镁过度混合→片剂软化/溶出延迟

阶段8: 压片
  → 片重调节/硬度调节
  → 决策点：压片压力是否合适？
  → ⚠️ 过硬→崩解超限；过软→碎裂
```

### 2.4 片剂质量控制

| 检查项目 | ChP通则 | 限度要求 | 检测方法 |
|----------|---------|----------|----------|
| 重量差异 | 0942 | ±5%（0.3g以上）/±7.5%（0.3g以下） | 逐片称重 |
| 崩解时限 | 0921 | 普通片≤15min/薄膜衣≤30min | 崩解仪 |
| 溶出度 | 0931 | Q值（通常≥70%或80%） | 溶出仪+UV/HPLC |
| 含量均匀度 | 0941 | A+2.2S≤15.0 | 逐片测定 |
| 硬度 | — | 3-8kg（一般） | 硬度仪 |
| 脆碎度 | 0961 | 失重≤1% | 脆碎度仪 |
| 含量 | 各论 | 95.0-105.0% | HPLC/UV/滴定 |

## 3. 注射剂制备实验

### 3.1 注射剂分类与要求
| 类型 | 灭菌方法 | 洁净级别 | 关键要求 |
|------|----------|----------|----------|
| 最终灭菌小容量 | 湿热灭菌121°C/15min | C级背景+B级灌装 | 无菌保证SAL≤10⁻⁶ |
| 最终灭菌大容量 | 湿热灭菌121°C/15min | C级背景+B级灌装 | 同上+细菌内毒素 |
| 非最终灭菌 | 无菌工艺 | B级背景+A级灌装 | 无菌工艺验证 |
| 冻干制剂 | 冻干+无菌灌装 | B级背景+A级 | 水分控制+无菌 |

### 3.2 注射剂配制工艺模拟

```
阶段1: 原辅料检验
  → 原料药/辅料/注射用水检验合格
  → 决策点：原料药是否符合注射用标准？

阶段2: 配液
  → 注射用水溶解→活性炭脱色→精滤
  → 决策点：活性炭用量是否合适（0.1-0.5%）？
  → pH调节（药典规定范围）
  → 决策点：pH是否在规定范围？
  → 等渗调节（氯化钠/葡萄糖）
  → 决策点：是否需要等渗调节？

阶段3: 过滤
  → 0.45μm→0.22μm两级过滤
  → 决策点：过滤顺序是否正确？
  → 过滤器完整性测试（起泡点测试）
  → 决策点：起泡点压力是否合格？

阶段4: 灌装
  → 灌装量=标示量+过量（2%）
  → 决策点：灌装精度是否合格？
  → 半压塞/全压塞

阶段5: 灭菌
  → 最终灭菌：121°C/15min（F0≥8）
  → 决策点：灭菌程序是否经验证？
  → ⚠️ 热敏性API不能用最终灭菌→无菌工艺

阶段6: 灯检
  → 可见异物检查（白点/纤维/玻璃屑）
  → 决策点：灯检标准是否执行？
  → ChP通则0904

阶段7: 质量检验
  → 无菌/内毒素/pH/含量/渗透压/可见异物
  → 决策点：所有项目是否合格？
```

### 3.3 注射剂常见问题
| 问题 | 原因 | 解决措施 |
|------|------|----------|
| 可见异物 | 环境/设备/操作污染 | 加强洁净区管理/过滤器检查 |
| pH不合格 | 配制计算错误/CO2影响 | 复核计算/惰性气体保护 |
| 含量偏低 | 吸附损失/降解 | 活性炭用量调整/稳定剂 |
| 无菌不合格 | 灭菌失败/密封缺陷 | 灭菌验证/密封完整性测试 |
| 内毒素超标 | 原料/水/设备污染 | 原料内毒素控制/注射用水监测 |

## 4. 稳定性试验

### 4.1 ICH Q1A(R2)稳定性试验要求

| 试验类型 | 条件 | 最短时间 | 适用 |
|----------|------|----------|------|
| 影响因素-高温 | 60°C | 10天 | 原料药+制剂 |
| 影响因素-高湿 | 25°C/92.5%RH | 10天 | 原料药+制剂 |
| 影响因素-强光 | 4500lx±500lx | 10天 | 原料药+制剂 |
| 加速试验 | 40°C±2°C/75%RH±5% | 6个月 | 原料药+制剂 |
| 中间条件 | 30°C±2°C/65%RH±5% | 12个月 | 制剂（加速显著变化时） |
| 长期试验 | 25°C±2°C/60%RH±5% | 12个月（注册） | 原料药+制剂 |

### 4.2 稳定性考察决策点
1. 包装选择：根据影响因素试验结果选择合适包装
2. 有效期制定：长期试验数据外推
3. 贮藏条件：根据稳定性数据确定
4. 复检期（原料药）：基于长期试验数据

## 5. GMP车间模拟

### 5.1 车间布局原则
- 人流物流分离
- 洁净级别递减（高→低）
- 防止交叉污染
- 气流组织（正压差：高洁净区→低洁净区）

### 5.2 环境监测
| 监测项目 | 频率 | 限度(D级) | 限度(C级) |
|----------|------|-----------|-----------|
| 悬浮粒子(≥0.5μm) | 每次使用前 | 3,520,000/m³ | 352,000/m³ |
| 悬浮粒子(≥5μm) | 每次使用前 | 29,000/m³ | 2,900/m³ |
| 沉降菌(4h) | 每次使用 | ≤200 cfu/皿 | ≤100 cfu/皿 |
| 浮游菌 | 每周 | ≤200 cfu/m³ | ≤100 cfu/m³ |
| 表面微生物 | 每周 | ≤50 cfu/25cm² | ≤25 cfu/25cm² |
| 人员微生物 | 每次出洁净区 | ≤50 cfu/手套 | ≤25 cfu/手套 |

### 5.3 批记录审核要点
- 批生产记录完整性
- 偏差记录与处理
- 中间体检验结果
- 成品检验结果
- 物料平衡
- 清场记录
- 环境监测记录
