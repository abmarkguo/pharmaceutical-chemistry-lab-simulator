## 2. 合成实验核心知识体系

### 2.1 反应机理教学要点

每个合成实验必须讲解：
1. **电子转移路径**：箭头推电子机理
2. **反应中间体**：碳正离子/碳负离子/自由基/卡宾
3. **立体化学**：构型保持/翻转/消旋化
4. **热力学vs动力学控制**：产物分布的影响因素
5. **副反应**：可能的竞争反应及抑制策略

### 2.2 试剂安全速查表

| 试剂 | GHS分类 | 主要危险 | 防护要求 | 应急措施 |
|------|---------|----------|----------|----------|
| 浓硫酸 | 腐蚀性1A | 严重灼伤/脱水 | 护目镜+耐酸手套+通风橱 | 皮肤接触：大量水冲洗15min+就医 |
| 乙酸酐 | 腐蚀性1B | 腐蚀/刺激呼吸道 | 通风橱+防毒面具 | 吸入：移至空气新鲜处 |
| 金属钠 | 遇水放出易燃气体 | 爆炸/火灾/灼伤 | 惰性气氛+干燥环境 | 灭火：干沙/NaCl，禁用水/CO2 |
| 二氯甲烷 | 致癌2B/生殖毒性 | 致癌/麻醉 | 通风橱+密闭操作 | 皮肤：肥皂水冲洗 |
| 甲醇 | 易燃/毒性 | 失明/中毒 | 通风橱+远离火源 | 吸入：移至新鲜空气+吸氧 |
| PCC(吡啶氯铬酸盐) | 致癌/氧化性 | 致癌/灼伤 | 通风橱+全面罩 | 皮肤：大量水冲洗 |
| n-BuLi | 自燃/腐蚀 | 自燃/灼伤 | 惰性气氛+低温+通风橱 | 灭火：干粉/干沙，禁用水 |
| THF(含过氧化物) | 易燃/过氧化物爆炸 | 爆炸/火灾 | 检测过氧化物+避光 | 灭火：泡沫/干粉 |

### 2.3 反应监控方法

| 方法 | 适用场景 | 优点 | 局限 |
|------|----------|------|------|
| TLC | 有机反应普遍适用 | 快速/直观/廉价 | 定量能力有限 |
| HPLC | 精确监控/含量测定 | 高精度/可定量 | 耗时/成本高 |
| GC | 挥发性物质/残留溶剂 | 高灵敏度 | 不适用热不稳定物 |
| NMR(反应内) | 机理研究/动力学 | 结构信息丰富 | 设备要求高 |
| IR(反应内) | 官能团变化监控 | 快速/实时 | 干扰因素多 |
| 质谱(在线) | 中间体鉴定 | 高灵敏度/结构推断 | 设备昂贵 |

### 2.4 分离纯化方法选择

```
产物分离纯化决策树：

水溶性产物？
├─ 是 → 水相体系
│   ├─ 萃取（水相→有机相：调pH/盐析）
│   └─ 重结晶（水或水-有机混合溶剂）
└─ 否 → 有机相体系
    ├─ 萃取（有机相→水相洗涤：去酸/碱/盐）
    ├─ 重结晶（选择合适溶剂）
    ├─ 柱层析（硅胶/氧化铝/反相）
    └─ 蒸馏（液体产物：常压/减压）

纯度不够？
├─ 重结晶不纯 → 换溶剂/活性炭脱色/二次重结晶
├─ 柱层析不纯 → 调整洗脱比例/换固定相/制备HPLC
└─ 蒸馏不纯 → 分馏/减压蒸馏/分子蒸馏
```

## 3. 经典合成实验教学方案

### 3.1 阿司匹林（乙酰水杨酸）合成

**反应原理**：
水杨酸 + 乙酸酐 → 乙酰水杨酸 + 乙酸
（酸催化酯化反应，浓硫酸或磷酸催化）

**关键教学点**：
- 酚羟基的酰化选择性（羧基不反应）
- 反应温度控制（85-90°C，过高导致副产物聚酯）
- 重结晶溶剂选择（乙醇-水体系）
- 纯度检测（FeCl3显色反应检查游离水杨酸）

**标准关联**：
- ChP二部：阿司匹林各论——含量≥99.0%，游离水杨酸≤0.1%
- USP：Aspirin各论——同上
- ICH Q3A：杂质鉴定阈值0.10%

**安全要点**：
- 乙酸酐：腐蚀性，遇水分解放热
- 浓硫酸：强腐蚀性，催化用量少（2-3滴）
- 反应放热：加料时控制滴加速度

**决策点设计**：
1. 催化剂选择：浓硫酸 vs 磷酸 vs 无催化？（安全性 vs 收率）
2. 反应温度：室温 vs 加热？（时间 vs 副产物）
3. 重结晶溶剂：纯乙醇 vs 乙醇-水 vs 乙醚-石油醚？（纯度 vs 收率）
4. 纯度检测：FeCl3显色 vs HPLC vs 熔点？（快检 vs 精确）

### 3.2 磺胺甲噁唑（SMZ）合成

**反应路线**（3步）：
1. 乙酰苯胺 → 对乙酰氨基苯磺酰氯（氯磺化）
2. 对乙酰氨基苯磺酰氯 + 3-氨基-5-甲基异噁唑 → 缩合
3. 水解脱乙酰基 → 磺胺甲噁唑

**关键教学点**：
- 多步合成的路线设计逻辑
- 保护基策略（乙酰基保护氨基）
- 氯磺化反应条件控制（温度/时间/氯磺酸用量）
- 缩合反应pH控制
- 水解反应条件（酸/碱水解）

**标准关联**：
- ChP二部：磺胺甲噁唑各论
- ICH Q7：原料药GMP要求

### 3.3 卡巴他赛半合成

**反应路线**（从10-DAB出发）：
1. 10-DAB保护（7,13-二保护）
2. 10位甲基化反应
3. 脱保护+侧链偶联
4. 卡巴他赛成品

**关键教学点**：
- 半合成的保护基策略
- 区域选择性反应（10位 vs 7位 vs 13位）
- 手性中心的保持
- 多步合成的总收率优化

**标准关联**：
- GMP附录2：原料药
- ICH Q11：起始物料选择

## 4. 合成实验渐进式模拟设计

### 4.1 模拟阶段与决策点

```
阶段1: 文献调研与路线选择
  决策点：选择合成路线（收率/成本/安全性/环保性考量）

阶段2: 试剂准备与安全评估
  决策点：高危试剂处理方式是否正确
  ⚠️ 安全提示：金属钠遇水爆炸/有机锂自燃

阶段3: 反应装置搭建
  决策点：装置选择是否匹配反应条件
  ⚠️ 安全提示：无水无氧操作需Schlenk线/手套箱

阶段4: 投料与反应启动
  决策点：加料顺序/温度控制/搅拌速率

阶段5: 反应监控
  决策点：TLC展开剂选择/反应终点判断

阶段6: 后处理
  决策点：淬灭方式/萃取溶剂选择/pH调节

阶段7: 分离纯化
  决策点：重结晶vs柱层析vs蒸馏

阶段8: 产物表征与收率计算
  决策点：表征方法选择/纯度判定标准
```

### 4.2 常见错误与反馈设计

| 阶段 | 常见错误 | 后果 | 反馈内容 |
|------|----------|------|----------|
| 投料 | 加料顺序错误 | 副反应/收率低 | 解释亲核试剂与亲电试剂的加料顺序原理 |
| 温控 | 温度过高 | 副产物/分解/爆炸 | 关联Arrhenius方程与副反应活化能 |
| 监控 | 反应未完全就停止 | 收率低 | 展示TLC板上残留原料斑点 |
| 萃取 | 分液漏斗操作不当 | 乳化/产物损失 | 讲解分液漏斗正确使用方法 |
| 干燥 | 干燥剂选择错误 | 产物污染/分解 | 对比不同干燥剂的适用范围 |
| 重结晶 | 溶剂选择不当 | 纯度差/收率低 | 讲解相似相溶原理与重结晶溶剂选择 |

## 5. 产率优化教学

### 5.1 影响因素分析
| 因素 | 影响机制 | 优化方向 |
|------|----------|----------|
| 反应温度 | 活化能/反应速率/副反应 | 温度窗口筛选 |
| 反应时间 | 转化率/副产物积累 | TLC/HPLC监控 |
| 摩尔比 | 转化率/成本/后处理难度 | 过量试剂选择 |
| 催化剂 | 反应速率/选择性 | 催化剂筛选/载量优化 |
| 溶剂 | 溶解度/反应活性/安全性 | 溶剂效应研究 |
| 浓度 | 分子内vs分子间反应 | 稀释/浓缩策略 |
| pH | 酸碱催化/产物稳定性 | 缓冲体系选择 |

### 5.2 DOE实验设计
- 因素筛选：Plackett-Burman设计
- 响应面优化：Box-Behnken/中心复合设计
- 关键响应：收率/纯度/杂质谱
- 统计分析：ANOVA/回归模型/预测R²
