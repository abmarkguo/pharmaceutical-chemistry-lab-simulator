# 药理/毒理实验指南 | Pharmacology & Toxicology Experiment Guide

## 1. 实验分类与设计

### 1.1 药理学实验类型
| 类型 | 研究目的 | 典型实验 | 周期 |
|------|----------|----------|------|
| 药效学 | 药物作用与疗效 | 镇痛实验/抗炎实验/降压实验 | 1-4周 |
| 药代动力学 | ADME过程 | 血药浓度-时间曲线/生物利用度 | 1-2周 |
| 安全药理 | 核心系统安全 | 心血管/呼吸/中枢神经系统 | 2-4周 |
| 作用机制 | 作用靶点与通路 | 受体结合/酶抑制/信号通路 | 4-12周 |

### 1.2 毒理学实验类型
| 类型 | ICH指导原则 | 研究目的 | 动物要求 |
|------|-------------|----------|----------|
| 急性毒性 | S4 | MTD/致死量 | 啮齿类 |
| 重复给药毒性 | S4 | 靶器官毒性/NOAEL | 啮齿类+非啮齿类 |
| 遗传毒性 | S2(R1) | 致突变/致染色体畸变 | 体外+体内 |
| 生殖毒性 | S5(R3) | 生育力/胚胎-胎仔/围产期 | 大鼠/兔 |
| 致癌性 | S1A/S1B/S1C | 长期致癌 | 大鼠/小鼠 |
| 安全药理 | S7A/S7B | 核心系统功能安全 | 大鼠/犬/猴 |
| 免疫毒性 | S8 | 免疫系统影响 | 大鼠/小鼠 |

## 2. 3R原则实施

### 2.1 Replacement（替代）
| 替代方法 | 适用场景 | 验证状态 |
|----------|----------|----------|
| 体外细胞实验 | 细胞毒性/肝毒性筛选 | 部分OECD验证 |
| 组织/器官灌注 | 心脏/肝脏/肾脏功能研究 | 研究方法 |
| 计算机模拟 | QSAR预测/ADME预测 | ICH认可辅助手段 |
| 器官芯片 | 多器官相互作用 | 前沿研究 |
| 低等生物 | 斑马鱼/线虫 | 部分验证 |

**决策点设计**：
- 该实验目的是否可以用体外方法替代？
- 如果可以替代，替代方法是否经过验证？
- 如果不能替代，选择最低等动物模型

### 2.2 Reduction（减少）
| 策略 | 实施方法 |
|------|----------|
| 样本量计算 | 统计功效分析确定最小样本量 |
| 实验设计优化 | 拉丁方设计/交叉设计减少个体差异 |
| 信息共享 | 文献数据引用/历史对照 |
| 终点优化 | 减少不必要的检测时间点 |
| 多器官/多终点 | 一只动物采集多组织/多指标 |

### 2.3 Refinement（优化）
| 策略 | 实施方法 |
|------|----------|
| 麻醉镇痛 | 手术操作使用适当麻醉/术后镇痛 |
| 丰容 | 改善饲养环境（垫料/玩具/群居） |
| 人道终点 | 提前终止标准（体重下降>20%/无法进食等） |
| 给药方式优化 | 选择最小痛苦途径 |
| 操作培训 | 规范操作减少应激 |

## 3. 药效学实验设计

### 3.1 经典药效学实验

#### 镇痛实验（热板法/醋酸扭体法）

**热板法**：
- 动物：雌性小鼠（18-22g）
- 仪器：热板仪（55±0.5°C）
- 指标：舔足反应潜伏期
- 分组：空白/阳性对照（吗啡）/受试药（低中高剂量）
- 决策点：温度设置是否正确（过高→烫伤，过低→不反应）

**醋酸扭体法**：
- 动物：小鼠
- 致痛剂：0.6%醋酸（0.1mL/10g ip）
- 指标：15min内扭体次数
- 阳性对照：阿司匹林/吗啡
- 决策点：醋酸浓度和注射体积是否准确

#### 抗炎实验（角叉菜胶足肿胀法）

- 动物：大鼠（150-200g）
- 致炎剂：1%角叉菜胶（0.1mL/足跖sc）
- 指标：足容积变化（0.5/1/2/4/6h）
- 阳性对照：吲哚美辛/地塞米松
- 决策点：测量时间点设计是否合理

### 3.2 实验设计原则

```
分组方案设计：
├─ 空白对照组（不给药不给模型）
├─ 模型对照组（给模型不给药）
├─ 阳性对照组（给模型+已知有效药）
├─ 受试药低剂量组
├─ 受试药中剂量组
└─ 受试药高剂量组

样本量计算（功效分析）：
  n = (Zα + Zβ)² × 2σ² / δ²
  α = 0.05, β = 0.20 (功效80%)
  σ = 预实验标准差
  δ = 预期差异

剂量设计：
  低剂量 = 预期有效量的1/3-1/2
  中剂量 = 预期有效量
  高剂量 = 预期有效量的2-3倍（或MTD的1/10）
```

### 3.3 量效关系分析
- 最小有效量(MED)
- 半数有效量(ED50)
- 最大效应(Emax)
- 效价强度(potency) vs 效能(efficacy)
- Bliss法/改良Karber法/Logit法计算ED50

## 4. 药代动力学实验

### 4.1 采血方案设计

```
单次给药PK采血设计：
├─ 吸收相：3-4个点（给药后5/15/30/60min）
├─ 分布相：2-3个点（1/2/4h）
├─ 消除相：3-4个点（8/12/24h或更远）
├─ 总采血点：8-12个
└─ 每点采血量：大鼠0.2-0.3mL（不超过总血量1%）

决策点设计：
├─ 采血途径选择（颈静脉插管/眼眶/尾尖）
├─ 采血量是否在动物耐受范围
├─ 采血时间点是否覆盖完整PK曲线
├─ 抗凝剂选择（肝素/EDTA/枸橼酸）
└─ 样品处理（离心条件/保存温度/冻融次数）
```

### 4.2 PK参数计算
| 参数 | 含义 | 计算方法 |
|------|------|----------|
| Cmax | 峰浓度 | 实测最大值 |
| Tmax | 达峰时间 | 对应Cmax的时间 |
| AUC0-t | 0到t时曲线下面积 | 梯形法 |
| AUC0-∞ | 0到无穷面积 | AUC0-t + Ct/λz |
| t1/2 | 半衰期 | 0.693/λz |
| CL | 清除率 | Dose/AUC |
| Vd | 表观分布容积 | Dose/(AUC×λz) |
| MRT | 平均驻留时间 | AUMC/AUC |
| F | 生物利用度 | (AUC_test/AUC_ref)×Dose_ref/Dose_test |

### 4.3 生物等效性
- 设计：交叉设计（制剂A→清洗期→制剂B）
- 评价指标：Cmax/AUC的90%置信区间
- 可接受标准：80.00%-125.00%
- 决策点：清洗期是否足够（≥5个t1/2）

## 5. 毒理学实验设计

### 5.1 急性毒性实验

**经典方法（已较少使用）**：
- LD50测定（改良Karber法/Bliss法）

**替代方法（推荐）**：
- 序贯法/上下法（OECD 425）
- 固定剂量法（OECD 420）
- 急性毒性阶级法（OECD 423）

**观察指标**：
- 死亡率/死亡时间
- 临床症状（给药后持续观察14天）
- 体重变化
- 大体解剖

**决策点**：
- 给药后观察频率（1h/4h/8h/24h/每日）
- 人道终点执行标准
- 大体解剖发现与给药的关联

### 5.2 重复给药毒性实验

| 项目 | 啮齿类 | 非啮齿类 |
|------|--------|----------|
| 恢复期 | 2-4周 | 2-4周 |
| 动物数/组 | 10/性别（主+恢复） | 4/性别（主+恢复） |
| 剂量组 | 低/中/高+对照 | 低/中/高+对照 |
| 给药途径 | 拟临床途径 | 拟临床途径 |
| 给药周期 | 依赖临床用药周期 | 依赖临床用药周期 |

**给药周期对应**：
| 临床用药 | 动物给药周期 |
|----------|-------------|
| 单次给药 | 2周 |
| ≤2周 | 1个月 |
| ≤1月 | 3个月 |
| ≤3月 | 6个月（啮齿类）/9个月（非啮齿类） |

### 5.3 安全药理核心组合
| 系统 | 实验 | 指标 | 指导原则 |
|------|------|------|----------|
| 中枢神经 | 功能观察组合(FOB)/Irwin's | 自主活动/行为/神经反射 | S7A |
| 心血管 | 遥测/体外hERG/体内心电图 | 血压/心率/QT/PR/QRS | S7A/S7B |
| 呼吸 | 呼吸功能测定 | 呼吸频率/潮气量/分钟通气量 | S7A |

- 电子数据有审计追踪
- 修改标注原因和操作者
- 禁止预填表格
- 禁止追溯性记录

## 7. 动物实验伦理审查

### 7.1 IACUC审查要点
1. 科学必要性（是否有替代方案）
2. 动物选择合理性（种属/数量/性别）
3. 痛苦最小化（麻醉/镇痛/人道终点）
4. 人员资质（培训/持证）
5. 饲养管理（环境/丰容/兽医护理）
6. 人道终点设定
7. 安乐死方法（AVMA指南）

### 7.2 人道终点标准
| 指标 | 终点标准 |
|------|----------|
| 体重 | 下降≥20%初始体重 |
| 体况评分 | BCS ≤ 2/5 |
| 进食饮水 | 完全停止≥24h |
| 肿瘤 | 直径>2cm（小鼠）/ >4cm（大鼠）或溃破 |
| 活动能力 | 无法自主进食/饮水/移动 |
| 呼吸 | 呼吸困难/持续呼吸频率异常 |
| 疼痛 | 对镇痛药无反应的持续疼痛 |
