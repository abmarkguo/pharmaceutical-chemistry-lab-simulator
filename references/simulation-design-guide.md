# 渐进式模拟设计指南 | Simulation Design Guide

## 1. 渐进式决策模拟方法论

### 1.1 核心理念

渐进式决策模拟（Progressive Decision-Based Simulation）是一种教学设计方法，将完整的实验流程分解为多个阶段，每个阶段设置决策点，学生通过选择/判断/输入与系统交互，系统提供即时反馈和知识锚点，实现"做中学、错中改"的教学效果。

### 1.2 设计原则

| 原则 | 含义 | 实现方式 |
|------|------|----------|
| 真实性 | 模拟接近真实实验 | 真实试剂/设备/参数/标准引用 |
| 安全性 | 虚拟环境允许试错 | 危险操作有安全提示/后果模拟 |
| 教育性 | 每步有明确学习目标 | 知识锚点/原理弹窗/标准引用 |
| 渐进性 | 从简单到复杂 | 基础→中级→高级难度分级 |
| 反馈性 | 即时反馈错误 | 后果描述/原理讲解/纠正建议 |
| 完整性 | 完整实验流程 | 从准备到报告的全流程覆盖 |

## 2. 模拟阶段设计

### 2.1 阶段分解原则

```
实验流程分解原则：

1. 按时间顺序分解
   → 严格遵循实验操作的时间序列
   → 不可跳过前置步骤

2. 按逻辑模块分解
   → 准备模块→操作模块→检测模块→报告模块
   → 模块间有明确的输入-输出关系

3. 每阶段3-5个操作步骤
   → 过多导致认知负荷过重
   → 过少导致流程碎片化

4. 关键阶段设置决策点
   → 安全关键点
   → 技术关键点
   → 知识关键点
   → 标准合规关键点
```

### 2.2 决策点类型

| 类型 | 交互方式 | 示例 | 教学目标 |
|------|----------|------|----------|
| 选择题 | 单选/多选 | "选择正确的加料顺序" | 概念理解/规范操作 |
| 判断题 | 对/错 | "此温度下是否可以开始加热" | 条件判断/安全意识 |
| 数值输入 | 输入数字 | "计算所需试剂用量" | 计算/配液 |
| 排序题 | 拖拽排序 | "排列正确的操作顺序" | 流程理解 |
| 匹配题 | 连线匹配 | "匹配试剂与其MSDS分类" | 知识关联 |
| 情景判断 | 情景选择 | "发现异常现象时如何处理" | 应急能力 |

### 2.3 反馈层级设计

```
反馈层级：

Level 0 - 正确反馈
  → "✓ 正确！" + 简要说明 + 进入下一阶段
  → 时长：即时

Level 1 - 提示反馈（轻微错误）
  → "⚠️ 再想想..." + 提示信息
  → 允许重新选择
  → 不扣分或轻微扣分

Level 2 - 纠正反馈（技术错误）
  → "✗ 不正确。" + 后果说明 + 正确做法 + 原理讲解
  → 重新操作（记录错误次数）
  → 扣分

Level 3 - 警告反馈（严重错误）
  → "🚨 安全警告！" + 严重后果描述 + 应急措施
  → 强制观看安全教学
  → 重新操作（扣分较多）
  → 记录为安全违规

Level 4 - 灾难反馈（危险操作）
  → "🚨🚨🚨 严重安全事件！" + 灾难性后果模拟
  → 实验中断/强制安全培训
  → 需教师解锁后继续
  → 严重扣分/实验不合格
```

### 3.2 模拟案例示例（阿司匹林合成-阶段3: 投料与反应启动）

```markdown
## 阶段3: 投料与反应启动

### 背景说明
酯化反应需要控制加料顺序和温度，以确保反应顺利进行并避免副反应。
本阶段是整个合成实验的核心步骤。

### 操作指引
1. 将水杨酸（2.0g）加入干燥的锥形瓶中
2. 加入乙酸酐（5.0mL）
3. 滴加浓硫酸（3滴）作为催化剂
4. 水浴加热至85-90°C
5. 反应15-20分钟

### ⚠️ 安全提示
- 乙酸酐：腐蚀性，遇水分解放热，在通风橱中操作
- 浓硫酸：强腐蚀性，催化用量少，切勿过量
- 反应放热：加料后先混匀再加热

### 决策点3.1: 加料顺序

**情景**: 你准备开始投料，三种试剂（水杨酸、乙酸酐、浓硫酸）应该按什么顺序加入？

**选项**:
- A. 先加乙酸酐，再加水杨酸，最后滴加浓硫酸
- B. 先加水杨酸，再加乙酸酐，最后滴加浓硫酸
- C. 先加浓硫酸，再加水杨酸，最后加乙酸酐
- D. 三者同时加入

**正确答案**: B
**错误反馈**:
- 选A → "不推荐。乙酸酐遇水杨酸表面微量水会分解放热，可能飞溅。
  正确做法：先加固体水杨酸，再加液体乙酸酐浸没，最后滴加催化剂。"
- 选C → "🚨 危险！浓硫酸直接加入空瓶会附着在瓶壁，后续加入有机物时
  会剧烈放热导致飞溅或爆沸。浓硫酸应最后滴加。"
- 选D → "✗ 混合加入无法控制反应速率，且浓硫酸可能局部浓度过高导致碳化。"

**知识弹窗**:
> **酯化反应机理**：水杨酸的酚羟基与乙酸酐发生亲核酰基取代反应。
> 浓硫酸作为质子酸催化剂，活化乙酸酐的羰基碳，使其更易受酚羟基氧的亲核攻击。
> 加料顺序原则：固体→液体→催化剂（少量）。

### 决策点3.2: 反应温度控制

**情景**: 反应混合物已准备好，你需要设置水浴温度。以下哪个温度范围最合适？

**选项**:
- A. 室温（20-25°C）
- B. 40-50°C
- C. 85-90°C
- D. 120°C以上

**正确答案**: C
**错误反馈**:
- 选A → "室温下反应极慢，可能需要数小时甚至无法完成。
  虽然不会产生安全问题，但效率太低。85-90°C是酯化反应的标准温度。"
- 选B → "温度偏低，反应速率较慢，可能需要1小时以上。
  可以尝试但不是最佳选择。"
- 选D → "🚨 危险！温度过高会导致：
  ① 乙酸酐大量挥发（沸点139°C，但在高温下蒸气压显著升高）
  ② 水杨酸升华损失
  ③ 副产物（聚合物/脱羧产物）增多
  ④ 可能引发火灾风险"

**知识弹窗**:
> **温度对酯化反应的影响**：
> - Arrhenius方程：温度升高10°C，反应速率约增加2-3倍
> - 但高温会加速副反应（如水杨酸自身缩合形成聚酯）
> - 85-90°C是平衡了反应速率和选择性的最佳温度
> - ChP要求阿司匹林中游离水杨酸≤0.1%，温度控制是关键因素

### 阶段小结
- 加料顺序：固体→液体→催化剂
- 反应温度：85-90°C水浴
- 催化剂用量：浓硫酸2-3滴（催化量）
- 反应时间：15-20分钟
```

## 4. 难度分级设计

### 4.1 三级难度体系

| 维度 | 基础级 | 中级 | 高级 |
|------|--------|------|------|
| 操作步骤 | 标准流程，无异常 | 含1-2个异常处理 | 多异常+开放性决策 |
| 决策点数量 | 3-5个 | 6-8个 | 9-12个 |
| 反馈深度 | 正确/错误+简述 | +原理讲解 | +方案对比+优化建议 |
| 安全事件 | 无 | 1个轻微事件 | 多事件+应急决策 |
| 数据处理 | 给定数据计算 | 原始数据处理 | 数据分析+统计推断 |
| 标准引用 | 标注ChP | ChP+ICH对比 | 多标准对比+合规判断 |
| 适合用户 | 本科低年级 | 本科高年级 | 研究生/教师 |

### 4.2 难度适配规则
```
用户输入学生层次 → 自动选择难度

本科低年级 → 基础级
  → 简化流程/多提示/少决策点/标准答案明确

本科高年级 → 中级
  → 完整流程/含异常/6-8决策点/需理解原理

研究生/教师 → 高级
  → 开放性设计/多异常/多标准对比/需优化决策
```

## 5. 评分系统设计

### 5.1 评分维度与权重

```
总分(100) = 标准规范度(40) + 安全意识(25) + 数据完整性(20) + 操作效率(15)

标准规范度(40分)
  = (正确步骤数 / 总步骤数) × 40
  - 每个错误步骤扣除相应分数
  - 严重错误额外扣分

安全意识(25分)
  = 25 - (危险操作次数 × 5 + 警告次数 × 2)
  - 最低0分
  - 灾难性事件直接0分

数据完整性(20分)
  = (有效数据点 / 预期数据点) × 20
  - 数据记录规范度 ±5分
  - 数据处理正确性 ±5分

操作效率(15分)
  = (标准时间 / 实际时间) × 15
  - 上限15分
  - 超时适当扣分
```

### 5.2 评分等级
| 总分 | 等级 | 描述 |
|------|------|------|
| 90-100 | 优秀 | 操作规范/安全意识强/数据完整/效率高 |
| 80-89 | 良好 | 基本规范/偶有错误/能纠正 |
| 70-79 | 合格 | 能完成实验/有较多错误/需改进 |
| 60-69 | 勉强合格 | 勉强完成/错误较多/需重点改进 |
| <60 | 不合格 | 无法完成/严重错误/安全违规 |

### 5.3 能力诊断雷达图

```
雷达图维度：
  ① 理论知识掌握（决策点正确率）
  ② 操作规范性（标准步骤执行率）
  ③ 安全意识（安全事件避免率）
  ④ 数据处理能力（数据记录和处理正确率）
  ⑤ 问题解决能力（异常处理正确率）
  ⑥ 标准合规能力（标准引用和执行正确率）

输出：六维雷达图 + 各维度详细分析 + 改进建议
```

## 6. 模拟实验质量检查清单

### 6.1 内容检查
- [ ] 实验概述完整（名称/类型/难度/时长/目标）
- [ ] 适用标准条款标注准确
- [ ] 化学反应方程式配平
- [ ] 试剂用量计算正确
- [ ] 操作步骤逻辑合理
- [ ] 安全提示全面准确

### 6.2 决策点检查
- [ ] 每个阶段至少1个决策点
- [ ] 决策选项有区分度（非显而易见）
- [ ] 错误反馈包含后果+原理+正确做法
- [ ] 安全相关决策点有⚠️标记
- [ ] 知识弹窗内容准确且相关

### 6.3 教学设计检查
- [ ] 难度与目标用户匹配
- [ ] 学习目标明确可测
- [ ] 反馈层次合理（Level 0-4）
- [ ] 评分系统公平合理
- [ ] 能力诊断维度完整
