## 1. 药物分析方法分类

### 1.1 按方法类型分类
| 类型 | 方法 | 原理 | 药典通则 |
|------|------|------|----------|
| 容量分析 | 酸碱滴定/氧化还原/配位/非水滴定 | 化学计量反应 | 0700 |
| 光谱分析 | UV-Vis/IR/荧光/原子吸收/拉曼 | 光与物质相互作用 | 0400 |
| 色谱分析 | HPLC/GC/TLC/离子色谱/分子排阻 | 分离+检测 | 0500 |
| 质谱分析 | LC-MS/GC-MS | 质荷比分离 | 0431 |
| 核磁共振 | 1H-NMR/13C-NMR | 核磁矩 | 0421 |
| 电化学分析 | pH/电位滴定/伏安法 | 电化学信号 | — |
| 生物学分析 | 效价测定/微生物限度/内毒素 | 生物反应 | 1100-1200 |

| 目的 | 分析项目 | 典型方法 |
|------|----------|----------|
| 鉴别 | 确认药品身份 | IR/UV/TLC/HPLC保留时间/化学反应 |

| 含量测定 | 主成分定量 | HPLC/UV/滴定 |
| 体内分析 | 生物样本药物浓度 | LC-MS/MS |

## 2. 仪器操作模拟要点

### 2.1 HPLC操作模拟

#### 开机与准备流程
```
1. 检查流动相（液面/脱气/有效期）
   → 决策点：流动相是否需要过滤脱气？
   → 错误反馈：未脱气导致基线噪声/泵压波动

2. 更换色谱柱
   → 决策点：色谱柱类型选择（C18/C8/苯基/HILIC）
   → 错误反馈：柱类型不匹配导致分离失败

3. 系统初始化
   → 开泵 → 排气泡 → 设流速 → 平衡
   → 决策点：流速设置是否合适？
   → 错误反馈：流速过高导致柱压超限

4. 系统适用性试验（SST）
   → 决策点：SST结果是否满足要求？
   → 理论塔板数 ≥ N要求
   → 拖尾因子 0.8-1.5
   → 重复性 RSD ≤ 2.0%（含量测定）/1.0%（杂质）
   → 分离度 ≥ 1.5（杂质检查）/2.0（含量）

5. 样品分析
   → 决策点：进样量是否在线性范围内？
   → 错误反馈：超载导致峰变形

6. 数据处理
   → 决策点：积分参数设置是否合理？
   → 错误反馈：积分起点/终点错误导致面积偏差
```

#### HPLC常见故障与排查
| 现象 | 可能原因 | 排查方法 |
|------|----------|----------|
| 柱压升高 | 柱头堵塞/管路堵塞 | 更换保护柱/冲洗 |
| 柱压波动 | 泵漏液/气泡 | 检查密封/排气 |
| 基线漂移 | 柱未平衡/温度变化 | 延长平衡/恒温 |
| 基线噪声 | 流动相污染/检测器灯能量低 | 更换流动相/换灯 |
| 峰拖尾 | 柱污染/死体积/硅羟基作用 | 换柱/减小死体积/加改性剂 |
| 峰分裂 | 柱头塌陷/进溶剂不匹配 | 换柱/改用流动相进样 |
| 保留时间漂移 | 柱温不稳/流动相变化 | 恒温/重新配制流动相 |

### 2.2 GC操作模拟

#### 关键参数设置
| 参数 | 设置原则 | 常见问题 |
|------|----------|----------|
| 进样口温度 | 比最高沸点组分高20-50°C | 温度过低导致歧视 |
| 柱温 | 根据组分沸程设定程序升温 | 初始温度过高导致分离差 |
| 载气流速 | 线速度20-40 cm/s（He）/30-50 cm/s（H2/N2） | 过高导致分离差 |
| 分流比 | 10:1-100:1（根据浓度调整） | 不分流模式用于痕量分析 |
| 检测器温度 | 比柱温高20-50°C | 过低导致冷凝 |

### 2.3 UV-Vis操作模拟

#### 定量分析流程
```
1. 基线校正（空白溶剂）
   → 决策点：空白溶剂选择是否正确？

2. 吸收光谱扫描
   → 确定最大吸收波长
   → 决策点：λmax是否与文献一致？

3. 标准曲线绘制
   → 5-7个浓度点
   → 决策点：线性范围是否覆盖样品浓度？
   → R² ≥ 0.999

4. 样品测定
   → 决策点：吸光度是否在0.2-0.8范围？
   → 错误反馈：超出线性范围需稀释

5. 含量计算
   → A = εbc（Beer-Lambert定律）
   → 决策点：计算是否正确？
```

### 2.4 滴定分析模拟

#### 滴定类型与指示剂
| 滴定类型 | 指示剂 | 终点颜色变化 | 适用药物 |
|----------|--------|-------------|----------|
| 酸碱滴定 | 酚酞 | 无色→粉红 | 阿司匹林/维生素C |
| 酸碱滴定 | 甲基红 | 红→黄 | 盐酸麻黄碱 |
| 非水滴定 | 结晶紫 | 紫→蓝→绿 | 弱碱性药物（生物碱） |
| 碘量法 | 淀粉 | 无色→蓝/蓝→无色 | 维生素C/青霉素 |
| 铈量法 | 邻二氮菲亚铁 | 红→淡蓝 | 硫酸亚铁/葡萄糖 |
| 亚硝酸钠法 | 永停法/中性红 | 电流突变/紫→无 | 磺胺类/普鲁卡因 |
| 配位滴定 | 铬黑T/二甲酚橙 | 酒红→蓝/红→黄 | 含金属药物 |

## 4. 典型药物分析实验

### 4.1 阿司匹林含量测定（酸碱滴定法）

**标准**：ChP2020二部 阿司匹林各论

**原理**：
阿司匹林（游离羧基）+ NaOH → 钠盐 + H2O

**操作流程**：
1. 精密称取阿司匹林约0.4g
2. 加中性乙醇（对酚酞指示液显中性）20mL溶解
3. 加酚酞指示液3滴
4. 用氢氧化钠滴定液(0.1mol/L)滴定至粉红色
5. 每1mL NaOH(0.1mol/L)相当于18.02mg C9H8O4

**决策点设计**：
1. 为什么用中性乙醇而非水溶解？（阿司匹林水中水解）
2. 为什么不直接用水做溶剂？（酯键水解副反应）
3. 终点判断：粉红色30秒不褪色

**安全提示**：
- NaOH溶液：腐蚀性，避免皮肤接触
- 乙醇：易燃，远离火源

### 4.2 维生素C注射液含量测定（碘量法）

**标准**：ChP2020二部 维生素C各论

**原理**：
维生素C（还原型）+ I2 → 脱氢抗坏血酸 + 2HI

**操作流程**：
1. 精密量取注射液适量（约相当于维生素C 50mg）
2. 加稀醋酸2mL（防止维生素C氧化）
3. 加淀粉指示液1mL
4. 用碘滴定液(0.05mol/L)滴定至蓝色
5. 每1mL I2(0.05mol/L)相当于8.806mg C6H8O6

**决策点设计**：
1. 为什么加稀醋酸？（酸性条件下维生素C更稳定，减少空气氧化）
2. 为什么要新鲜配制？（维生素C易氧化）
3. 注射液中的抗氧剂（亚硫酸氢钠）会干扰吗？（会消耗碘，需加丙酮掩蔽）

### 4.3 HPLC法测定对乙酰氨基酚片含量

**标准**：ChP2020二部 对乙酰氨基酚各论

**色谱条件**：
- 色谱柱：C18柱（4.6mm×250mm，5μm）
- 流动相：甲醇-水（40:60）
- 检测波长：254nm
- 流速：1.0mL/min
- 柱温：30°C
- 进样量：10μL

**操作流程**：
1. SST合格后进样
2. 对照品溶液和供试品溶液分别进样
3. 按外标法以峰面积计算含量
4. 含量应为标示量的95.0%-105.0%

**决策点设计**：
1. 流动相比例对保留时间的影响
2. 检测波长选择依据（UV扫描确定λmax）
3. 柱温对分离的影响
4. 进样精密度（RSD ≤ 1.0%）

## 5. OOS调查模拟

### 5.1 OOS调查流程
```
步骤1: 实验室调查（第一阶段）
  ├─ 复核计算
  ├─ 复核原始数据
  ├─ 检查仪器状态
  └─ 重新进样（如可行）

步骤2: 扩大调查（如第一阶段未找到原因）
  ├─ 重新取样
  ├─ 重新制备样品
  ├─ 方法适用性检查
  └─ 人员/设备/环境排查

步骤3: 生产调查（如实验室排除OOS）
  ├─ 生产记录审查
  ├─ 偏差回顾
  └─ 取样代表性评估

步骤4: 结论与处置
  ├─ OOS确认 → 不放行
  ├─ OOS推翻（实验室错误）→ 原结果无效，重新检验
  └─ 无明确结论 → 风险评估决定
```

### 5.2 OOS决策点设计
- 计算错误：复核发现→纠正→记录
- 仪器故障：确认→维修→重新校准→重新分析
- 样品制备错误：重新制备→重新分析
- 方法问题：方法偏差调查→验证补充
- 生产原因：启动生产偏差调查

## 6. 数据完整性与合规

### 6.1 ALCOA+原则
| 原则 | 含义 | 实践要求 |
|------|------|----------|
| Attributable | 可归属 | 操作者/审核者签名 |
| Legible | 可读 | 永久性记录，不可擦除 |
| Contemporaneous | 同时性 | 操作时即时记录 |
| Original | 原始性 | 保留原始数据 |
| Accurate | 准确性 | 真实反映实际情况 |
| Complete | 完整性 | 包括所有数据（含失败数据） |
| Consistent | 一致性 | 符合SOP和时序逻辑 |
| Enduring | 持久性 | 整个保存期内可读 |
| Available | 可用性 | 审计时可调阅 |

### 6.2 禁止行为
- 修改电子数据不留下审计追踪
- 删除原始数据
- 重复进样至合格（"进到合格为止"）
- 手工修改数据不标注原因
- 未授权人员操作仪器
- 共享账号密码
