# 实验报告撰写指南 | Lab Report Writing Guide

## 1. 实验报告结构

### 1.1 标准结构
```
实验报告
├─ 1. 实验名称
├─ 2. 实验目的
├─ 3. 实验原理
├─ 4. 试剂与仪器
├─ 5. 实验步骤
├─ 6. 数据与结果
├─ 7. 讨论与分析
├─ 8. 结论
├─ 9. 参考文献
└─ 10. 附录（如有）
```

### 1.2 各部分撰写要求

#### 1. 实验名称
- 格式：[药物名称/实验类型] [实验内容]
- 示例："阿司匹林的合成与纯度检查"
- 避免照抄讲义标题，用自己的语言概括

#### 2. 实验目的
- 用动词开头：掌握/测定/比较/验证/制备
- 2-4条，覆盖知识/技能/素养
- 示例：
  - 掌握酯化反应合成阿司匹林的原理和操作方法
  - 学习重结晶纯化技术和熔点测定方法
  - 了解ChP对阿司匹林的质量要求

#### 3. 实验原理
- 核心反应方程式（必须配平、标注条件）
- 反应机理简述（关键步骤，不展开全机理）
- 方法原理（如果是分析实验）
- 标准依据（ChP/USP条款）
- **常见问题**：照抄教材、没有自己的理解、原理与目的脱节

#### 4. 试剂与仪器
- 试剂表格：名称/规格/用量/用途
- 仪器表格：名称/型号/关键参数
- 标注特殊要求（如干燥/无水/新开封）

**示例表格**：
| 试剂名称 | 规格 | 用量 | 用途 |
|----------|------|------|------|
| 水杨酸 | AR，138.12 | 2.0g | 原料 |
| 乙酸酐 | AR，102.09 | 5.0mL | 酰化剂 |
| 浓硫酸 | AR | 3滴 | 催化剂 |

#### 5. 实验步骤
- **用自己的语言描述**，不要照抄SOP
- 标注关键操作和注意事项
- 按时间顺序记录实际操作（非计划操作）
- 记录异常情况和处理措施
- 示例：
  ```
  1. 称取水杨酸2.003g于干燥的100mL锥形瓶中
     （注意：使用万分之一天平，精确至0.001g）
  2. 加入乙酸酐5.0mL，振摇使水杨酸溶解
  3. 滴加浓硫酸3滴，轻轻摇匀
     （注意：在通风橱中操作，浓硫酸具有强腐蚀性）
  4. 水浴加热至88°C，保温反应18min
     （实际温度86-90°C波动，使用温度计监控）
  ```

#### 6. 数据与结果
- 原始数据表格化
- 有效数字规范
- 计算过程完整展示
- 图表规范（图题在下，表题在上）

**有效数字规则**：
```
加减法：结果以小数位数最少的数据为准
  12.35 + 0.123 = 12.47（保留2位小数）

乘除法：结果以有效数字最少的数据为准
  2.50 × 3.1 = 7.8（保留2位有效数字）

对数：尾数位数与真数有效数字位数相同
  log(1.23×10⁻⁴) = -3.910（3位尾数对应3位有效数字）
```

**修约规则**（四舍六入五成双）：
- 4以下舍去：2.134 → 2.13
- 6以上进一：2.136 → 2.14
- 5后面有数进一：2.1351 → 2.14
- 5后面无数看前位：2.135 → 2.14（前位3为奇数进一）
- 5后面无数看前位：2.145 → 2.14（前位4为偶数不进）

#### 7. 讨论与分析
这是报告的核心部分，占分最高。

**讨论框架**：
```
7.1 结果分析
  → 与理论值/文献值对比
  → 偏差分析（如收率偏低/纯度不够）

7.2 误差来源分析
  → 系统误差（仪器/方法/试剂）
  → 偶然误差（操作/环境）
  → 具体分析每个误差对结果的影响方向和大小

7.3 异常现象分析
  → 记录实验中观察到的异常
  → 分析可能原因
  → 说明处理措施

7.4 改进建议
  → 针对误差来源提出具体改进
  → 针对操作提出优化建议
  → 针对方法提出替代方案

7.5 理论关联
  → 将实验结果与理论原理关联
  → 讨论影响因素的理论解释
  → 联系实际应用（药品质量/临床用药）
```

**常见问题**：
- ❌ 讨论空洞："本次实验比较成功，学到了很多"
- ❌ 照抄教材：重复原理部分
- ❌ 无数据支撑：讨论与实际数据无关
- ❌ 无改进建议：只分析问题不提出对策
- ✅ 有理有据：用数据说话，关联理论
- ✅ 深入思考：分析误差传递，量化影响
- ✅ 有建设性：提出可行改进方案

#### 8. 结论
- 2-3句话简洁总结
- 直接回应实验目的
- 有数据支撑
- 示例：
  ```
  本实验成功合成了阿司匹林，收率78.5%，熔点135-137°C
  （ChP规定熔点135-138°C），FeCl3检查呈阴性（游离水杨酸
  <0.1%），产品符合ChP质量要求。
  ```

#### 9. 参考文献
- 格式：GB/T 7714 或 ACS Style
- 至少引用3篇（教材/药典/文献）
- 示例：
  ```
  [1] 国家药典委员会. 中华人民共和国药典(2020年版二部)[M]. 
      北京: 中国医药科技出版社, 2020: 45-46.
  [2] 钱清华, 荆丽艳. 药物合成技术[M]. 北京: 化学工业出版社, 
      2019: 85-92.
  [3] ICH. Q3A(R2) Impurities in New Drug Substances[S]. 2006.
  ```

## 2. 数据可视化规范

### 2.1 表格规范
- 表号表题在表格**上方**
- 表格用三线表（无竖线）
- 表头标注单位
- 数据对齐（小数点对齐）
- 示例：

```
表1 阿司匹林合成实验数据

  序号    水杨酸(g)    乙酸酐(mL)    反应温度(°C)    产量(g)    收率(%)
  ─────────────────────────────────────────────────────────────
   1      2.003        5.0           88             2.38       75.2
   2      2.001        5.0           87             2.45       77.6
   3      1.998        5.0           89             2.50       79.1
  ─────────────────────────────────────────────────────────────
  平均    2.001±0.003  5.0           88±1           2.44±0.06  77.3±2.0
```

### 2.2 图表规范
- 图号图题在图**下方**
- 坐标轴标注：量名（单位），如"浓度（mg/L）"
- 刻度等间距（对数坐标除外）
- 数据点用符号标记，趋势线用线连接
- 图例标注清晰
- 示例：

```
          浓度(mg/L)
    100 ─ ─ ─ ─ ─ ─ ─ ─
     80 ─ ─ ● ─ ─ ─ ─ ─
     60 ─ ●─ ─ ● ─ ─ ─ ─
     40 ─ ─ ─ ─ ●─ ─ ─ ─
     20 ─ ─ ─ ─ ─ ─ ● ─
      0 ─ ─ ─ ─ ─ ─ ─ ●─
        ────────────────── 时间(h)
        0   2   4   6   8

图1 阿司匹林血药浓度-时间曲线
```

### 2.3 常见图表类型
| 图表类型 | 适用场景 | 示例 |
|----------|----------|------|
| 折线图 | 趋势变化/时间序列 | 反应进程监控/溶出曲线 |
| 柱状图 | 离散数据比较 | 不同条件收率比较 |
| 散点图 | 相关性分析 | 标准曲线 |
| 饼图 | 比例分布 | 杂质谱分布（少用） |
| 误差条图 | 均值±标准差 | 多组数据比较 |

## 3. 常见错误诊断

### 3.1 数据处理常见错误

| 错误类型 | 典型表现 | 正确做法 |
|----------|----------|----------|
| 有效数字错误 | 2.5+3.14=5.64 | 2.5+3.14=5.6（保留1位小数） |
| 修约错误 | 2.35→2.4（四舍五入） | 2.35→2.4（四舍六入五成双，3为奇数进一） |
| 单位遗漏 | "浓度=0.5" | "浓度=0.5 mg/mL" |
| 单位混用 | "500mg + 0.3g" | "500mg + 300mg" 或 "0.5g + 0.3g" |
| 百分比计算 | (实测/理论)×100漏乘100 | (2.38/3.16)×100=75.3% |
| 平均值计算 | 未去除异常值直接平均 | Grubbs检验后去除异常值再平均 |
| RSD计算 | (SD/Mean)×100漏乘100或用SD | RSD=(SD/Mean)×100% |

### 3.2 讨论分析常见问题

| 问题 | 表现 | 改进方法 |
|------|------|----------|
| 空洞无物 | "实验很成功" | 用具体数据说明：收率78.5%，熔点合格 |
| 无误差分析 | "有一定误差" | 量化分析：温度波动±2°C导致收率波动约±3% |
| 无异常处理 | 忽略异常数据 | 记录并分析：第3批颜色偏黄，可能因... |
| 无改进建议 | 无 | 提出具体方案：改用磷酸催化可提高安全性 |
| 无理论关联 | 无 | 关联Arrhenius方程解释温度影响 |
| 照抄教材 | 重复原理 | 用自己的理解分析实验现象 |

### 3.3 格式常见问题
- 字体不统一（中英文混用不一致）
- 行距过密（建议1.5倍行距）
- 页码缺失
- 图表无编号
- 参考文献格式混乱
- 化学式未用下标/上标（如 H2O 应为 H₂O）

## 4. 实验报告自评清单

### 4.1 提交前自检

**内容完整性**：
- [ ] 实验目的明确，覆盖知识/技能/素养
- [ ] 原理部分有反应方程式和机理简述
- [ ] 试剂仪器表格完整，标注规格和用量
- [ ] 实验步骤用自己的语言描述，标注关键操作
- [ ] 原始数据完整记录，表格化呈现
- [ ] 数据处理过程完整，公式和计算步骤清晰
- [ ] 图表规范，有编号和标题
- [ ] 讨论深入，有误差分析和改进建议
- [ ] 结论简洁，回应实验目的
- [ ] 参考文献格式规范

**数据规范**：
- [ ] 有效数字正确（加减法/乘除法规则）
- [ ] 修约规则正确（四舍六入五成双）
- [ ] 单位完整无遗漏
- [ ] 误差/偏差正确表示（SD/RSD/置信区间）
- [ ] 图表坐标轴标注量和单位

**科学严谨性**：
- [ ] 无数据编造或修改
- [ ] 异常数据有分析和处理
- [ ] 讨论有数据支撑
- [ ] 结论基于实际数据
- [ ] 引用标注来源

**格式规范**：
- [ ] 字体/行距统一
- [ ] 页码完整
- [ ] 图表编号连续
- [ ] 化学式格式正确（下标/上标）
- [ ] 参考文献格式一致

### 4.2 质量自评等级
| 等级 | 标准 | 预估分数 |
|------|------|----------|
| A-优秀 | 全部满足，讨论深入有创见 | 90-100 |
| B-良好 | 基本满足，讨论较深入 | 80-89 |
| C-合格 | 大部分满足，讨论一般 | 70-79 |
| D-需改进 | 多项不满足，讨论空洞 | <70 |

## 5. 不同实验类型报告特点

### 5.1 合成实验报告
- 重点：反应方程式/收率计算/纯度检测/副产物分析
- 特有部分：合成路线设计逻辑/反应条件优化讨论
- 标准：ChP各论/ICH Q3A杂质控制

### 5.2 分析实验报告
- 重点：分析方法原理/系统适用性/数据计算/合规判定
- 特有部分：方法验证参数/标准曲线/精密度
- 标准：ChP通则/ICH Q2(R2)

### 5.3 药剂学实验报告
- 重点：处方设计/工艺流程/质控结果/稳定性
- 特有部分：处方筛选依据/GMP合规分析
- 标准：ChP通则09xx/GMP/ICH Q1A

### 5.4 药理实验报告
- 重点：实验设计/给药方案/数据统计/药效评价
- 特有部分：3R原则说明/伦理审查/统计分析方法
- 标准：GLP/ICH S系列/3R原则
