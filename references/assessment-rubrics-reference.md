# 考核评估量规 | Assessment Rubrics Reference

## 1. 考核体系架构

### 1.1 三阶段考核模型

```
实验考核体系
├─ 预习考核（20%）
│  ├─ 原理理解
│  ├─ 试剂安全
│  ├─ 操作流程
│  └─ 标准规范
├─ 操作考核（50%）
│  ├─ 准备工作
│  ├─ 操作规范
│  ├─ 安全意识
│  ├─ 数据记录
│  └─ 实验清理
└─ 报告考核（30%）
   ├─ 数据处理
   ├─ 结果分析
   ├─ 讨论深度
   └─ 规范格式
```

### 1.2 考核层级
| 层级 | 目的 | 题型 | 时间 |
|------|------|------|------|
| 预习考核 | 检查理论准备 | 选择/填空/判断/简答 | 20min |
| 操作考核 | 评估实验技能 | 操作评分点+安全检查 | 实验全程 |
| 报告考核 | 评价分析能力 | 报告评分量规 | 课后 |

### 2.2 命题规范

#### 单选题命题模板
```
[题干：描述一个实验情景或概念]
A. [干扰项1——常见误解]

  A → [为什么这个常见误解是错的]
  C → [为什么部分正确但不完全]
  D → [混淆了什么概念]
标准关联：[ChP/GMP/ICH条款]
```

#### 简答题命题模板
```
[题干：要求阐述原理或分析安全风险]

评分标准（12分）：
  内容要点（8分）：
    - 要点1（2分）：[具体内容]
    - 要点2（2分）：[具体内容]
    - 要点3（2分）：[具体内容]
    - 要点4（2分）：[具体内容]

  逻辑表达（2分）：
    - 条理清晰/逻辑严密（2分）
    - 基本清晰（1分）

  标准引用（2分）：
    - 正确引用标准条款（2分）
    - 提及标准但不准确（1分）
    - 未引用标准（0分）
```

### 2.3 题库示例（阿司匹林合成实验）

#### 单选题
```
Q1. 阿司匹林合成中使用浓硫酸的作用是：
A. 反应物，参与酯化反应
B. 脱水剂，吸收反应生成的水
C. 催化剂，质子化活化乙酸酐的羰基
D. 溶剂，溶解水杨酸

答案：C
解析：浓硫酸作为质子酸催化剂，提供H⁺质子化乙酸酐的羰基氧，
使羰基碳更亲电，便于水杨酸酚羟基氧的亲核攻击。
浓硫酸不作为反应物（A错）、不作为脱水剂（B错，反应中水量极少）、
不作为溶剂（D错，乙酸酐兼作溶剂和反应物）。
标准关联：有机化学酯化反应机理
```

#### 简答题
```
Q5. 阿司匹林合成实验中，如果反应温度过高（超过100°C），
可能产生哪些后果？请从反应机理和产品质量两个角度分析。（12分）

评分标准：
内容要点（8分）：
  - 乙酸酐挥发损失（2分）：乙酸酐沸点139°C，高温蒸气压增大
  - 水杨酸升华（2分）：水杨酸70°C以上开始升华
  - 副反应增多（2分）：高温促进水杨酸分子间缩合形成聚酯副产物
  - 产品质量下降（2分）：游离水杨酸超标（ChP要求≤0.1%）
逻辑表达（2分）
标准引用（2分）：引用ChP游离水杨酸限度
```

## 3. 操作考核评分量规

### 3.1 通用操作评分点

| 评分点 | 分值 | 评分标准 |
|--------|------|----------|
| 实验准备 | 5 | 试剂齐备/设备检查/PPE穿戴/通风橱启动 |
| 称量操作 | 5 | 天平校准/称量纸使用/称量精度/记录规范 |
| 试剂取用 | 5 | 取用顺序/用量准确/瓶盖归位/标签朝上 |
| 仪器操作 | 10 | 装置搭建/参数设置/操作顺序/使用规范 |
| 反应控制 | 10 | 温度控制/搅拌速度/反应监控/终点判断 |
| 分离纯化 | 10 | 方法选择/操作规范/效果评估/收率控制 |
| 数据记录 | 5 | 及时记录/格式规范/原始数据/修改标注 |
| 安全操作 | 5 | 通风橱使用/PPE全程/废液分类/紧急处理 |
| 实验清理 | 5 | 设备清洗/台面整理/废液处理/试剂归位 |
| **合计** | **60** | — |

> 注：操作考核满分60分，占实验总成绩50%（折算后满分50分）

### 3.2 四级评分标准

| 等级 | 描述 | 分值范围 |
|------|------|----------|
| 优秀 | 操作完全规范，无需指导，能处理异常 | 90-100% |
| 良好 | 操作基本规范，偶有小错能自行纠正 | 80-89% |
| 合格 | 能完成操作，有部分错误需提醒 | 70-79% |
| 不合格 | 操作不规范，需多次指导或存在安全隐患 | <70% |

### 3.3 安全一票否决项

以下情况操作考核直接不合格：
- ⛔ 未穿戴PPE进行实验操作
- ⛔ 在通风橱外操作挥发性/有毒试剂
- ⛔ 将试剂直接倒入水槽
- ⛔ 违反消防规定（明火/吸烟）
- ⛔ 擅自操作未经培训的高危设备
- ⛔ 发生安全事故未报告
- ⛔ 危险化学品管理违规

## 4. 报告考核评分量规

### 4.1 报告评分维度

| 维度 | 分值 | 评分要点 |
|------|------|----------|
| 实验目的与原理 | 5 | 目的明确/原理阐述准确/不照抄 |
| 试剂与仪器 | 5 | 规格型号完整/条件参数记录 |
| 实验步骤 | 5 | 自己语言描述/关键操作标注/不照抄SOP |
| 数据记录 | 10 | 原始数据完整/有效数字/单位/表格规范 |
| 数据处理 | 10 | 计算过程完整/公式正确/误差分析 |
| 图表制作 | 5 | 坐标标注/图题表题/刻度合理/格式规范 |
| 结果与讨论 | 10 | 结果陈述清晰/讨论有深度/异常分析/改进建议 |
| 结论 | 5 | 简洁明确/回应目的/有数据支撑 |
| 格式规范 | 5 | 字体/行距/页眉页脚/参考文献格式 |
| **合计** | **60** | — |

> 注：报告考核满分60分，占实验总成绩30%（折算后满分30分）

### 4.2 报告评分等级标准

#### 数据处理评分细则（10分）
| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 计算完整准确/有效数字正确/误差分析深入/统计方法正确 | 9-10 |
| 良好 | 计算基本正确/有效数字偶有错误/有误差分析但不够深入 | 7-8 |
| 合格 | 计算有部分错误/有效数字不规范/缺乏误差分析 | 5-6 |
| 不合格 | 计算严重错误/数据编造/无有效数字意识 | <5 |

#### 讨论分析评分细则（10分）
| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 深入分析误差来源/异常数据合理处理/有具体改进建议/联系理论 | 9-10 |
| 良好 | 有误差分析/能处理异常数据/改进建议较笼统 | 7-8 |
| 合格 | 误差分析表面化/异常数据未处理/改进建议空泛 | 5-6 |
| 不合格 | 无讨论/照抄教材/讨论与数据无关 | <5 |

## 6. 题库管理

### 6.1 题目元数据
每道题目应包含以下元数据：
```yaml
题目ID: [唯一编号]
实验项目: [关联实验]
题型: [单选/多选/填空/判断/简答/案例]
难度: [基础/中级/高级]
知识点: [关联知识点列表]
标准关联: [ChP/GMP/ICH条款]
认知层次: [记忆/理解/应用/分析/评价/创造]
使用次数: [统计字段]
正确率: [统计字段]
最后更新: [日期]
```

### 6.2 题库更新规则
- 每学期更新≥20%题目
- 正确率<30%的题目需修订
- 正确率>95%的题目考虑提高难度
- 新增标准/法规变化时更新相关题目
- 鼓励教师提交新题目（需审核）
