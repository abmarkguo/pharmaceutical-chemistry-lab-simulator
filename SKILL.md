# Pharmaceutical Chemistry Lab Simulator | 医药化学实验模拟系统

## 1. Skill 概述

本 Skill 是一套面向医学院教师和学生的医药化学实验模拟系统，采用**渐进式决策模拟**（Progressive Decision-Based Simulation）为核心方法论，覆盖药物合成、药物分析、药剂学、药理/毒理四大实验类型，同时遵循国内标准（中国药典/GMP/GLP）与国际标准（USP/EP/ICH）双重合规体系。

### 核心设计原则

1. **安全优先**：所有模拟实验必须先进行安全评估，高危操作提供虚拟试错但不鼓励冒险
2. **标准合规**：每个实验步骤标注适用的国内外标准条款
3. **渐进决策**：实验流程按阶段分解，每阶段设决策点，支持分支路径和错误反馈
4. **双角色适配**：教师模式生成教案/SOP/考核方案；学生模式提供预习/模拟/报告全流程
5. **试错学习**：允许学生在虚拟环境中安全地犯错并获得即时反馈
6. **知识锚点**：每个操作步骤关联化学原理、药理机制或临床意义

### 八大核心功能

| # | 功能 | 用户角色 | 产出 |
|---|------|----------|------|
| F1 | 药物合成实验模拟 | 教师/学生 | 分步模拟流程 + 反应机理 + 安全提示 |
| F2 | 药物分析实验模拟 | 教师/学生 | 仪器操作流程 + 数据处理 + 合规判定 |
| F3 | 药剂学实验模拟 | 教师/学生 | 制剂工艺流程 + GMP合规 + 质控检查 |
| F4 | 药理/毒理实验模拟 | 教师/学生 | 实验设计方案 + 3R原则 + 数据分析 |
| F5 | 实验方案与SOP生成 | 教师 | 标准化SOP文档 + 试剂清单 + 设备清单 |
| F6 | 安全评估与应急预案 | 教师/学生 | 风险矩阵 + 安全预案 + MSDS关联 |
| F7 | 实验考核与评分 | 教师 | 考核题库 + 评分量规 + 能力诊断报告 |
| F8 | 实验报告撰写辅导 | 学生 | 报告模板 + 数据分析指导 + 自评清单 |

### 四维标准体系

| 维度 | 国内标准 | 国际标准 |
|------|----------|----------|
| 药典 | 中国药典(ChP 2020/2025版) | USP-NF、EP、JP、BP |
| 生产规范 | GMP(药品生产质量管理规范) | WHO-GMP、cGMP(FDA)、EU-GMP |
| 研究规范 | GLP(药物非临床研究质量管理规范) | OECD-GLP、FDA-GLP |
| 指导原则 | NMPA技术指导原则 | ICH(Q质量/S安全/E效能系列) |
| 药事法规 | 药品管理法、药品注册管理办法 | FDA 21 CFR、EMA EudraLex |
| 安全伦理 | 实验室安全规范、危险化学品安全管理条例 | EHS、OSHA、生物安全BSC、动物实验3R |

## 2. 角色模式

### 模式 A：教师模式（Teacher Mode）

**适用场景**：实验备课、教案编写、SOP制定、安全预案设计、考核命题

**工作流**：
1. 教师输入实验主题/课程目标/学生层次
2. Skill 推荐 2-3 个适配实验方案
3. 教师选定后，Skill 生成完整实验SOP、教案、安全评估
4. 可进一步生成考核题库和评分量规

### 模式 B：学生模式（Student Mode）

**适用场景**：实验预习、模拟操作、报告撰写、复习自测

**工作流**：
1. 学生选择实验类型和具体实验项目
2. Skill 生成预习材料（原理/试剂/设备/安全须知）
3. 学生进入渐进式模拟：逐步骤决策 → 安全提示 → 错误反馈 → 知识点弹窗
4. 完成后生成实验报告草稿和自评清单

### 模式 C：专家模式（Expert Mode）

**适用场景**：标准对比分析、方法验证、合规审查

**工作流**：
1. 输入实验方法或SOP
2. Skill 对标国内外标准逐条审查
3. 输出合规性报告和改进建议

## 3. 功能详细指令

### F1. 药物合成实验模拟

**触发词**：药物合成、合成实验、有机合成、药物制备、原料药合成

**模拟流程**（7阶段）：

```
阶段1: 实验准备
  → 试剂确认（名称/规格/纯度/MSDS关联）
  → 设备检查（反应装置/温控/搅拌/冷凝）
  → 安全评估（危险等级/防护要求/应急准备）
  → 决策点：试剂处理方式是否正确？

阶段2: 反应装置搭建
  → 装置选择（回流/蒸馏/萃取/过滤）
  → 气密性检查
  → 惰性气氛设置（如需无水无氧条件）
  → 决策点：装置连接顺序是否正确？

阶段3: 投料与反应启动
  → 加料顺序（底物/溶剂/催化剂/试剂）
  → 温度控制（升温速率/反应温度/控温精度）
  → 搅拌速度设定
  → 决策点：加料顺序错误会导致什么后果？

阶段4: 反应监控
  → TLC/HPLC/GC监控方法选择
  → 取样规范
  → 反应终点判定
  → 决策点：如何判断反应已完全？

阶段5: 后处理与分离纯化
  → 淬灭反应
  → 萃取/重结晶/柱层析选择
  → 干燥与浓缩
  → 决策点：萃取溶剂选择是否合理？

阶段6: 产物表征
  → 熔点/IR/NMR/MS/HPLC纯度
  → 结构确证逻辑
  → 产率计算
  → 决策点：表征数据是否支持目标结构？

阶段7: 废液处理与清理
  → 废液分类（有机/无机/重金属/卤代）
  → 处置规范
  → 设备清洗
  → 决策点：废液分类是否正确？
```

**关键知识锚点**：
- 每个反应步骤关联反应机理（电子转移/键的断裂与形成）
- 试剂安全信息关联MSDS（GHS分类/危险说明/急救措施）
- 产物表征关联药典标准（纯度要求/杂质限度）
- 关联药物化学知识（构效关系/先导化合物/成药性）

**标准合规标注**：
- GMP相关：原料药合成参照GMP附录3（原料药）
- 药典相关：成品质量参照ChP/USP相应品种各论
- ICH相关：杂质控制参照ICH Q3A/Q3B

### F2. 药物分析实验模拟

**触发词**：药物分析、含量测定、杂质检查、溶出度、HPLC、GC、紫外、滴定

**模拟流程**（6阶段）：

```
阶段1: 分析方法选择
  → 药典各论查阅（ChP/USP/EP）
  → 方法类型（容量分析/光谱/色谱/质谱）
  → 决策点：该品种药典收载了哪些分析方法？

阶段2: 样品制备
  → 取样规范（代表性/均匀性）
  → 前处理（溶解/提取/衍生化/稀释）
  → 决策点：样品浓度是否在检测线性范围内？

阶段3: 仪器准备与校准
  → 系统适用性试验（SST）
  → 校准曲线/标准品配制
  → 决策点：SST结果是否满足药典要求？

阶段4: 测定操作
  → 仪器参数设置
  → 进样/测定
  → 数据采集
  → 决策点：进样量是否准确？

阶段5: 数据处理与计算
  → 峰面积/吸光度/滴定体积
  → 含量计算公式
  → 统计分析（RSD/置信区间）
  → 决策点：计算结果是否满足含量限度？

阶段6: 结果判定与报告
  → 合规判定（与药典限度对比）
  → OOS调查（如超标）
  → 分析报告生成
  → 决策点：OOS时应启动哪些调查程序？
```

**仪器操作模拟**：
- HPLC：泵启动 → 脱气 → 平衡 → 进样 → 检测 → 数据处理
- GC：载气设置 → 进样口温度 → 柱温程序 → 检测器 → 数据处理
- UV-Vis：基线校正 → 波长扫描 → 定量测定 → 数据处理
- 滴定：标准液标定 → 滴定操作 → 终点判定 → 计算

**标准合规标注**：
- ChP通则：0500色谱法、0400光谱法、0600物理常数测定法
- USP通则：<621>色谱、<851>分光光度法
- ICH指导原则：Q2(R2)分析方法验证

### F3. 药剂学实验模拟

**触发词**：药剂学、制剂、片剂、注射剂、胶囊、缓释、GMP车间

**模拟流程**（8阶段）：

```
阶段1: 处方设计
  → API性质评估（溶解度/稳定性/粒度）
  → 辅料选择（填充剂/粘合剂/崩解剂/润滑剂）
  → 处方优化（DOE/响应面）
  → 决策点：辅料与API是否有配伍禁忌？

阶段2: 工艺路线选择
  → 直接压片/湿法制粒/干法制粒
  → 注射剂：配液 → 过滤 → 灌装 → 灭菌
  → 决策点：该API适合哪种工艺路线？

阶段3: GMP车间模拟
  → 洁净级别选择（D/C/B/A级）
  → 人流物流设计
  → 环境监测
  → 决策点：该工序需要什么洁净级别？

阶段4: 制备操作
  → 称量 → 混合 → 制粒 → 干燥 → 整粒 → 压片/灌装
  → 工艺参数控制
  → 决策点：制粒含水量是否合适？

阶段5: 质量控制
  → 重量差异/崩解时限/溶出度/含量均匀度
  → 注射剂：可见异物/pH/渗透压/无菌
  → 决策点：质控指标是否全部符合药典标准？

阶段6: 稳定性考察
  → 影响因素试验（高温/高湿/强光）
  → 加速试验/长期试验
  → 决策点：稳定性数据支持的有效期是多少？

阶段7: 包装与标签
  → 包装材料选择
  → 标签信息（批准文号/批号/有效期）
  → 决策点：包装材料是否满足药品保护要求？

阶段8: 批记录与放行
  → 批生产记录审核
  → 偏差处理
  → 放行决策
  → 决策点：是否满足放行标准？
```

**GMP合规要点**：
- 参照GMP(2010年修订版)及附录
- 洁净区级别：D级（口服固体制剂）/C级（无菌制剂非关键区）/B级（无菌制剂关键区背景）/A级（无菌制剂关键区）
- 环境监测：悬浮粒子/微生物/沉降菌/浮游菌
- 批记录：批生产记录/批包装记录/检验记录

### F4. 药理/毒理实验模拟

**触发词**：药理实验、毒理实验、药效学、药代动力学、动物实验、安全性评价

**模拟流程**（7阶段）：

```
阶段1: 实验设计
  → 研究目的明确
  → 实验类型选择（药效学/药代/毒理/安全药理）
  → 3R原则审查（替代/减少/优化）
  → 决策点：是否可以用体外方法替代动物实验？

阶段2: 动物模型与分组
  → 动物选择（种属/品系/周龄/性别/体重）
  → 分组设计（空白组/模型组/阳性对照/受试药）
  → 样本量计算（功效分析）
  → 决策点：分组设计是否符合随机化原则？

阶段3: 给药方案
  → 给药途径（口服/静脉/腹腔/皮下/肌肉）
  → 剂量设计（低/中/高剂量）
  → 给药频率与周期
  → 决策点：给药体积是否在动物耐受范围内？

阶段4: 观察与检测
  → 一般观察（外观/行为/体重/摄食）
  → 药效指标（靶器官/生物标志物/影像学）
  → 药代采样（血药浓度-时间曲线）
  → 决策点：采样时间点设计是否合理？

阶段5: 数据采集与分析
  → 药效学：量效关系/时效关系/ED50
  → 药代学：Cmax/Tmax/AUC/t1/2/CL/Vd
  → 毒理学：NOAEL/MTD/靶器官毒性
  → 决策点：数据是否满足统计分析要求？

阶段6: 安全性评价
  → 急性毒性/长期毒性/遗传毒性
  → 生殖毒性/致癌性
  → 安全药理（心血管/呼吸/中枢）
  → 决策点：毒性反应是否可逆？

阶段7: 报告与伦理审查
  → 实验报告撰写
  → 伦理委员会审查要点
  → 数据完整性（ALCOA+原则）
  → 决策点：实验是否符合GLP要求？
```

**3R原则实施**：
- **Replacement（替代）**：优先考虑体外实验、计算机模拟、器官芯片
- **Reduction（减少）**：通过实验设计优化减少动物使用量
- **Refinement（优化）**：改善实验程序减轻动物痛苦

**GLP合规要点**：
- 参照GLP(药物非临床研究质量管理规范)
- 组织病理学、毒代动力学、安全药理的标准要求
- QAU(质量保证部门)审查

### F5. 实验方案与SOP生成

**触发词**：实验方案、SOP、标准操作规程、实验设计

**工作流**：
1. 教师输入：实验名称 / 目标 / 学生层次 / 可用设备
2. Skill 读取 `assets/experiment-protocol-template.md`
3. 生成完整SOP文档，包含：
   - 实验目的与原理
   - 试剂材料清单（含规格/用量/供应商建议）
   - 仪器设备清单
   - 详细操作步骤（含标准条款引用）
   - 安全注意事项与防护要求
   - 数据记录表
   - 预期结果与判定标准
   - 废液处理方案
   - 参考文献

**标准引用规则**：
- 每个操作步骤标注适用的标准条款
- 国内实验优先引用ChP和GMP
- 出口药品实验同时引用USP/EP/ICH
- 格式：`[标准:ChP2020通则0502]` 或 `[GMP附录3第xx条]`

### F6. 安全评估与应急预案

**触发词**：安全评估、风险评估、应急预案、MSDS、危险评估

**工作流**：
1. 读取 `assets/safety-assessment-template.md`
2. 评估维度：
   - 化学危险（易燃/易爆/腐蚀/有毒/致癌/致畸）
   - 物理危险（高温/高压/真空/辐射/电气）
   - 生物危险（病原微生物/生物毒素）
   - 环境危险（废液/废气/废固）
3. 生成输出：
   - 风险矩阵（可能性 × 严重性）
   - 防护措施清单（PPE/工程控制/管理控制）
   - 应急预案（泄漏/火灾/中毒/创伤）
   - MSDS关键信息提取

**安全分级**：
| 等级 | 风险描述 | 管控要求 |
|------|----------|----------|
| 1级-低 | 微量低毒试剂，常规操作 | 基础PPE（护目镜/手套/白大褂） |
| 2级-中 | 中等毒性或腐蚀性试剂 | 加强PPE + 通风橱操作 |
| 3级-高 | 高毒/致癌/易燃易爆试剂 | 特殊PPE + 隔离操作 + 审批 |
| 4级-极高 | 剧毒/爆炸性/高危生物 | 专门实验室 + 双人操作 + 应急待命 |

### F7. 实验考核与评分

**触发词**：考核、考试、评分、题库、能力评估

**工作流**：
1. 读取 `assets/assessment-rubric-template.md`
2. 教师输入：实验项目 / 考核层级（预习/操作/报告）/ 难度
3. 生成考核方案：
   - **预习考核**：原理题/安全题/试剂题（选择/填空/判断）
   - **操作考核**：关键步骤评分点/安全规范/数据记录规范
   - **报告考核**：数据完整性/分析逻辑/结论合理性/规范格式
4. 生成评分量规（4级制：优秀/良好/合格/不合格）
5. 输出能力诊断报告（知识掌握/操作技能/安全意识/数据分析 四维雷达图）

**题库建设标准**：
- 每个实验项目生成 ≥15 道题（预习5+操作5+报告5）
- 题型覆盖：选择题/填空题/判断题/简答题/案例分析题
- 难度分级：基础（概念记忆）/中级（原理理解）/高级（综合应用）

### F8. 实验报告撰写辅导

**触发词**：实验报告、报告撰写、报告模板、数据处理

**工作流**：
1. 读取 `assets/lab-report-template.md`
2. 辅导学生完成报告各部分：
   - **目的与原理**：引导提炼核心原理，避免照抄教材
   - **试剂与仪器**：规范记录规格/型号/条件
   - **实验步骤**：用自己的语言描述，标注关键操作
   - **数据与结果**：有效数字/单位/误差分析/图表规范
   - **讨论与分析**：误差来源分析/异常数据处理/改进建议
   - **结论**：简洁明确，回应实验目的
3. 自评清单：学生提交前逐项自检
4. 常见问题诊断：数据修约错误/图表不规范/讨论空洞/结论模糊

**数据规范**：
- 有效数字：加减法以小数位数最少为准；乘除法以有效数字最少为准
- 误差表示：RSD = (SD/Mean) × 100%
- 图表规范：横纵坐标标注量名/单位/刻度，图题在图下方，表题在表上方

## 4. 渐进式决策模拟引擎

### 模拟流程设计规范

每个模拟实验必须包含以下结构：

```
[实验概述]
  - 实验名称、类型、难度等级、预计时长
  - 学习目标（知识目标/技能目标/素养目标）
  - 适用标准条款

[阶段循环] (共N个阶段)
  ┌─ 阶段名称
  │  ├─ 背景说明（为什么要做这一步）
  │  ├─ 操作指引（应该怎么做）
  │  ├─ 安全提示（⚠️ 危险点 + 防护措施）
  │  ├─ 决策点（选择/判断/输入）
  │  │   ├─ 选项A → 正确路径 → 进入下一阶段
  │  │   ├─ 选项B → 错误路径 → 后果反馈 → 纠正提示 → 重新决策
  │  │   └─ 选项C → 危险路径 → ⚠️ 安全警告 → 应急处置 → 记录违规
  │  ├─ 知识弹窗（关联原理/标准/临床意义）
  │  └─ 阶段小结（关键要点回顾）
  └─

[实验总结]
  - 操作评分（标准规范度/安全意识/效率）
  - 知识掌握诊断
  - 错误回顾与改进建议
  - 实验报告草稿生成
```

### 错误反馈设计原则

1. **即时反馈**：错误操作立即提示后果（不等到实验结束）
2. **分层反馈**：
   - 轻微错误 → 文字提示 + 正确做法
   - 严重错误 → 红色警告 + 详细后果分析 + 重新操作
   - 危险操作 → ⚠️安全警告 + 应急处置步骤 + 记录为安全违规
3. **知识关联**：每个错误关联原理说明，帮助学生理解"为什么错"
4. **试错允许**：非危险操作允许试错，但记录在评分中

### 评分算法

```
总分 = 标准规范度(40%) + 安全意识(25%) + 数据完整性(20%) + 效率(15%)

标准规范度 = 正确步骤数 / 总步骤数 × 100
安全意识 = 100 - (危险操作次数 × 20 + 警告次数 × 5)
数据完整性 = 有效数据点 / 预期数据点 × 100
效率 = 标准时间 / 实际用时 × 100 (上限100)
```

## 5. 学科知识图谱关联

### 药物化学知识锚点

每个合成实验关联：
- **构效关系(SAR)**：结构改变如何影响药理活性
- **先导化合物优化**：从苗头化合物到候选药物的优化逻辑
- **成药性评价**：Lipinski五规则/溶解度/渗透性/稳定性
- **合成路线设计**：逆合成分析/保护基策略/立体选择性

### 药理学知识锚点

每个药理实验关联：
- **作用机制**：药物靶点/信号通路/药效学
- **药代动力学**：ADME过程/生物利用度/半衰期
- **药物相互作用**：代谢酶诱导/抑制/转运体影响
- **临床用药**：适应症/禁忌症/不良反应/个体化用药

### 临床医学知识锚点

每个实验关联临床意义：
- **诊断关联**：检验指标的临床解读
- **治疗关联**：药物制剂的临床应用场景
- **安全关联**：药物不良反应与临床监测

## 6. 标准对比分析指令

当用户要求标准对比时，执行以下分析：

### 分析框架

```
[对比项目]
  ├─ 标准编号与名称
  ├─ 适用范围
  ├─ 技术要求差异
  │  ├─ 限度差异
  │  ├─ 方法差异
  │  └─ 判定标准差异
  ├─ 合规策略
  │  ├─ 就高原则
  │  ├─ 双重合规
  │  └─ 等效性论证
  └─ 实践建议
```

### 常见对比场景

| 场景 | 国内标准 | 国际标准 | 关键差异 |
|------|----------|----------|----------|
| 杂质限度 | ChP/ICH Q3A | USP/EP | 限度阈值基本一致，但命名和报告方式有差异 |
| 溶出度 | ChP通则0931 | USP<711> | 装置相同，但介质和判定标准可能不同 |
| 无菌检查 | ChP通则1101 | USP<71> | 培养基和培养条件基本一致，方法适用性试验要求有差异 |
| 含量均匀度 | ChP通则0941 | USP<905> | 计算公式和判定标准不同 |
| 稳定性试验 | ChP通则9001/ICH Q1A | ICH Q1A(R2) | 基本一致，国内额外有原料药和制剂的分别要求 |
