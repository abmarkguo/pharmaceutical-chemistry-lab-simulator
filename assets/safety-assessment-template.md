## 2. 风险评估矩阵

### 2.1 风险等级判定
```
严重性 →
可       │        │        │        │
能       │        │        │        │
性       │        │        │        │
↓        │        │        │        │
         ────────────────────────────
         轻微     中等     严重     灾难性

可能性：
  1-极低  2-低    3-中等   4-高    5-很高
严重性：
  1-轻微  2-中等  3-严重   4-非常严重 5-灾难性

风险等级 = 可能性 × 严重性
  1-4  → 1级（低风险）- 可接受
  5-9  → 2级（中风险）- 需控制措施
  10-15 → 3级（高风险）- 需特殊防护+审批
  16-25 → 4级（极高风险）- 需替代方案或取消
```
